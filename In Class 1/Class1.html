Step 1

  1. Open a web browser of your choice.
  2. Go to GitHub.com and create an account.
  3. Once account created, click the button to create a New Repository (see image below).

Step 2

Next, you should name your repository, select a few other parameters, and move to the next page.
Then you should select the Clone or download button and select Open in Desktop (see image).

Step 3

Clicking that button should automatically open your GitHub Desktop app. Locate where the files are saved on your comuter locally and add two directories (folders) to your GitHub project (see image below).

Step 4

Go back to your GitHub Desktop app, and commit your changed files to master, followed by pushing them from the Repository menu drop down.
